# Results

Outputs of the experiment campaign.

| Directory     | Versioned | Contents                                                  |
| ------------- | --------- | --------------------------------------------------------- |
| `runs/`       | no        | raw per-run logs; regenerated by the runner, git-ignored  |
| `aggregated/` | yes       | aggregated tables derived from the raw runs               |
| `tables/`     | yes       | final result tables for the thesis                        |
| `figures/`    | yes       | final figures                                             |

The raw `runs/` directory is intentionally git-ignored: at the scale of the full
matrix it is large and fully reproducible from the configurations and seeds. It
is distributed separately as part of the replication package.
